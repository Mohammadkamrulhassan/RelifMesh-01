\section{Key User Flows}

\subsection{Flow 1 --- Register Household}
Open App $\rightarrow$ New Household $\rightarrow$ Auto-capture GPS $\rightarrow$ Fill Form (name, NID, family size, flags) $\rightarrow$ Take Photo $\rightarrow$ Save $\rightarrow$ (Online: sync / Offline: queue) $\rightarrow$ Confirmation

\subsection{Flow 2 --- Log Distribution}
Search Household $\rightarrow$ Select $\rightarrow$ Duplicate Check $\rightarrow$ (Proceed / Override with reason) $\rightarrow$ Select Item \& Quantity $\rightarrow$ Photo + GPS $\rightarrow$ Submit

\subsection{Flow 3 --- Public Dashboard}
Open URL (no login) $\rightarrow$ View summary cards $\rightarrow$ View map with markers $\rightarrow$ Click marker for breakdown $\rightarrow$ Apply filters

\subsection{Flow 4 --- Calculate Need}
Select Ward $\rightarrow$ Auto-calculate from demographics $\times$ Sphere rates $\rightarrow$ View item-level breakdown $\rightarrow$ (Optional: Override) $\rightarrow$ Save $\rightarrow$ Heatmap updates

\subsection{Flow 5 --- Pledge Source}
View Heatmap $\rightarrow$ Identify under-served ward $\rightarrow$ Declare Pledge $\rightarrow$ Select area, item, quantity $\rightarrow$ Submit $\rightarrow$ Pledge visible on map

\section{Design System}

\subsection{Color Palette (Result09 Design System)}
\begin{itemize}[nosep]
    \item \textbf{Light Theme:} Primary \#2563eb, Surface \#ffffff, Text \#1e293b, Danger \#ef4444, Warning \#f59e0b, Success \#22c55e
    \item \textbf{Dark Theme:} Surface \#0f172a, Card \#1e293b, Text \#f1f5f9
    \item \textbf{Sidebar:} Dark panel with \#0f172a background, \#3b82f6 active indicator
\end{itemize}

\subsection{Typography}
Inter font family (300--900 weights), App Title 22px/700, Section Header 18px/600, Body 16px/400, Caption 13px/400

\subsection{Component Standards}
\begin{itemize}[nosep]
    \item Buttons: min height 40px, border-radius 8px, 6 variants
    \item Input fields: min height 44px, clear label, error/hint/success states
    \item Cards: 12px border-radius, 1px border, subtle shadow
    \item Data tables: full-width, striped rows, sticky header
    \item Sidebar: fixed dark panel, collapsible, mobile overlay
    \item Topbar: breadcrumbs, theme toggle, sync status indicator
\end{itemize}
