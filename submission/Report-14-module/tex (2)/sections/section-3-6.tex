\section{Offline-First Architecture}
\textbf{Strategy:} Single-database offline-first pattern using localforage (IndexedDB) in the browser.

\textbf{Conflict Resolution:} Last-write-wins by timestamp; conflicting versions saved in `sync\_conflicts` collection for manual review.

\section{Map \& Heatmap Architecture}
The map uses a layered rendering stack: Base Tiles (OpenStreetMap) $\rightarrow$ GeoJSON Layer (boundary polygons) $\rightarrow$ Heatmap Overlay (leaflet.heat WebGL) $\rightarrow$ Marker/Info Control (click interaction).

\textbf{Drill-down flow:} District $\rightarrow$ Upazila $\rightarrow$ Union $\rightarrow$ Ward.

\textbf{Heatmap color scheme:} Green (served/need met) $\rightarrow$ Yellow (partially served) $\rightarrow$ Red (unmet/critical).
